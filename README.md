# Healthcare Database Project

A simple SQL project simulating a healthcare database with patients, doctors, consultations, and exams.  
Built for learning and portfolio purposes.

## Technologies
- PostgreSQL

## Files
- `create_tables.sql`: creates the database structure
- `insert_data.sql`: inserts sample data
- `select_examples.sql`: displays tables and data

## Concepts Covered
- Table creation with primary and foreign keys  
- Data insertion and relationships  
- Data visualization with SELECT

## How to Use
1. Open pgAdmin or your PostgreSQL terminal.  
2. Create a database (example: `CREATE DATABASE healthcare_db;`)  
3. Run `create_tables.sql`  
4. Run `insert_data.sql`  
5. Run `select_examples.sql` to view results
